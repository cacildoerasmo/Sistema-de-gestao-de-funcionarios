# Sistema de gestão de funcionários 

Sistema Web, para gestão de funcionários. 

# Tecnologias e Ferramentas usadas
 * PHP
 * HTML
 * CSS
 * Bootstrap
 * Javascript
 * MySQL
 * Xampp
 
# Como usar
